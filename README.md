# unsplash-image-request
implement the search images using React JS and React Query


##Ingredients

- React JS (version 18)
- TypeScript
- React Query
- Unsplash API
- SCSS

##Installation

1. clone the repo
```
https://github.com/mark-mansfield/unsplash-image-request.git
```

2. Install dependencies

```
npm install
```

3. Run the project

```
   npm dev
```
